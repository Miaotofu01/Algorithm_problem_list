# A青春常数

$N=2026202520242023$ , $x + y = N$ , $0 \leq x < y$

易得 $x$ 的取值范围为 $[0,(N-1)/2]$ , 共 $(N-1)/2+1$ 种取法

```cpp
#include<bits/stdc++.h>
using namespace std;
using LL = long long;
#define int LL
void solve(){
    cout << (2026202520242023 / 2) + 1 << endl;
}
signed main(){
    int T = 1;
    // cin >> T;
    while(T--){
        solve();
    }
}
```
# C循环右移

只有数组全素全部相同时才可满足右移不变的条件，故答案为区间 $[X,Y]$ 的大小


```cpp
#include<bits/stdc++.h>
using namespace std;
using LL = long long;
#define int LL
void solve(){
    int n, x, y;
    cin >> n >> x >> y;
    cout << max(0LL, y - x + 1) << endl;
}
signed main(){
    int T = 1;
    cin >> T;
    while(T--){
        solve();
    }
}
```
# D 蓝桥竞技

观察题目易得
- 总人数一定是5的倍数
- 每个种类的最大人数一定是 总人数/5

只要满足上述条件，一定可以通过以下方式分配完成：
- 每次仅选取当前剩余人数最多的五个种类成团

故仅需检查以上两个条件即可

```cpp
#include<bits/stdc++.h>
using namespace std;
using LL = long long;
#define int LL
void solve(){
    int n;
    cin >> n;
    vector<int> a(n);
    for (int &e : a)
        cin >> e;
    int sum = accumulate(a.begin(),a.end(), 0LL);
    if (sum % 5 != 0)
    {
        cout << "F" << endl;
        return;
    }
    int maxcnt = sum / 5;
    for (int e : a)
    {
        if (e > maxcnt)
        {
            cout << "F" << endl;
            return;
        }
    }
    cout << "T" << endl;
}
signed main(){
    int T = 1;
    cin >> T;
    while(T--){
        solve();
    }
}
```
# E LQ聚合

最优策略必定是前面放一部分L，后面放一部分R，不可能是LR相间地放
故采用枚举策略
- 先假设全设置为R，计算答案
- 依次枚举"当前位置及以前设置为L，以后设置为R"的答案，对于每次枚举：
	- 减去当前位置作为R的贡献
	- 加上当前位置作为L的贡献
	- 即可完成一次更新
- 记录枚举出的最优解即可
```cpp
#include<bits/stdc++.h>
using namespace std;
using LL = long long;
#define int LL
void solve(){
    /*
    一定是前面放L,后面放R，从左到右枚举（当前位置以前放L,以后放R）的答案即可
    */
    int n;
    cin >> n;
    string s;
    cin >> s;
    int ans = 0; 
    int cntR = 0;
    for (int i = s.size() - 1; i >= 0; i--) // 假设全放R的答案
    {
        if (s[i] == 'L')
            ans += cntR;
        else
            cntR++;
    }
    int res = ans;
    for (int i = 0, cntL = 0; i < s.size(); i++)
    {
        if (s[i] == '?')
        {
            ans -= cntL; // 减去这个位置原先作为R的贡献
            ans += --cntR; // 加上这个位置作为L的贡献
            res=max(res,ans);
        }
        cntL += s[i] != 'R';
    }
    cout << res << endl;
}
signed main(){
    int T = 1;
    // cin >> T;
    while(T--){
        solve();
    }
}
```
